# استخدام الذكاء الاصطناعي

استخدمت AI أثناء بناء المشروع، لكن مش كبديل عن التجربة.

AI ساعد في:

- فهم كيف ملفات المشروع مربوطة ببعض
- شرح FastAPI routes وكيف تستدعي السكربتات
- كتابة وتعديل Bash scripts
- debug أخطاء NetworkManager
- شرح `tcpdump`
- تحسين طريقة الداشبورد
- فصل ملفات الواجهة عن `main.py`
- تجهيز repo public بدون أسرار
- كتابة التوثيق

الأشياء اللي بقيت علي:

- أجرب على Raspberry Pi الحقيقي
- أشوف شو فعلا يشتغل على الجهاز
- أقرر كيف لازم الداشبورد يتصرف
- أوفر setup الشبكة الحقيقي
- أتأكد إنه ما في بيانات خاصة تنرفع

المشروع لسه مشروعي. AI كان مساعد في الكود والفهم.
