# AI Assistance

I used AI during this project, but not as a replacement for testing.

The AI helped with:

- understanding how the project files were connected
- explaining FastAPI routes and how they call scripts
- writing and fixing Bash scripts
- debugging NetworkManager errors
- explaining `tcpdump`
- improving the dashboard flow
- splitting the UI files from `main.py`
- preparing a public GitHub repo without secrets
- writing documentation

What I still had to do:

- test on the Raspberry Pi
- check what actually worked on the device
- decide how the dashboard should behave
- provide the real network setup
- make sure no private data was published

The project is still mine. AI was used as a coding and learning assistant.
