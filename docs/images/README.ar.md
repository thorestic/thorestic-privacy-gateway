# الصور

الصور الموجودة:

- `dashboard-preview.png`: screenshot من واجهة الداشبورد الحقيقية وهي شغالة محليا.
- `network-page.png`: screenshot من صفحة Network الحقيقية وهي شغالة محليا.

قبل النشر، ممكن تضيف صورة حقيقية للجهاز:

- `hardware-setup.jpg`: صورة Raspberry Pi/router setup.

لا تعرض passwords، proxy settings حقيقية، أسماء شبكات خاصة، أسماء أجهزة شخصية، MAC addresses، logs، أو public IP addresses داخل الصور.
